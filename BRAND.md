# BRAND.md - Michael Loya Studio

## Color Palette (dark → light)

| Role | Hex | Name | Usage |
|------|-----|------|-------|
| **Primary Dark** | `#0a0a0a` | Black | Dark backgrounds, primary text on light bg |
| **Supporting** | `#3d2b1f` | Espresso | Secondary headings, body text on light bg, dividers |
| **Neutral** | `#8a8580` | Warm Gray | Muted text, subtitles, secondary UI elements |
| **Primary Light** | `#E8E0D4` | Parchment | Light backgrounds, cards, containers |
| **Accent** | `#ff8e3d` | Brand Orange | Links, hover states, active indicators — use intentionally |

## Usage Rules

### Orange (`#ff8e3d`)
- Links default to orange
- Hover states, active nav, interactive highlights
- Accent borders on contact buttons
- Should feel warm and intentional, not overwhelming

### Parchment (`#E8E0D4`)
- Primary light background — replaces pure white everywhere
- Cards, overlays, light-themed pages

### Black (`#0a0a0a`)
- Dark backgrounds (P1, die page dark transitions)
- Primary headings on light backgrounds
- Not pure `#000` — slightly softer

### Warm Gray (`#8a8580`)
- Subtitle text, timestamps, secondary info
- Muted UI elements (disabled states, placeholders)
- "Studio" text under logo

### Espresso (`#3d2b1f`)
- Body text on light backgrounds
- Secondary headings, descriptions
- Dividers, borders, grounding accents
- Pairs with parchment for a tonal, earthy feel
